  1. di dalam htdocs, mesti ada folder si5a2122o, sehingga pada browser bisa diakses dengan http://localhost/si5a2122o
  2. jika object not found (404), maka cek dulu nama folder antara htdocs dengan yang diketik di localhost
  3. jika ada tulisan error establishing database connection, cek dulu apakah mysql sudah running pada xampp control panel
  4. jika sudah running, buka wp-config.php pada folder si5a2122o, dan cek db_name. db name ini pastikan ada di phpmyadmin
  5. jika pada phpmyadmin belum ada nama databasenya, maka import dulu file sql bahan pertemuan ke 2 (hasil pertemuan ke 1)
  6. wordpress memiliki 2 sisi, 1 tampilan backend (untuk admin/dashboard) dan 1 nya lagi tampilan front end (untuk pengunjung/client area)
  7. untuk backend, harus login dulu di alamat http://localhost/si5a_dhita/wp-admin
  8. Pada menu pengguna, kita masuk ke profil yang mana tujuannnya untuk mengubah profil atau warna backend dan pasword untuk kita login
  9. pada menu profil kita dapat mengubah dan menambah nama pengguna kita lebih bagus
  10. pada saat kita membuat pengguna baru jika peranannya adalah pelanggan, maka orang tersebut cuma bisa update profil dan pasword saja tanpa bisa mengubah apapun
  11. pada saat kita membuat pengguna baru jika peranannya adalah kontributor, maka orang tersebut cuma bisa melihat komentar, peralatan tapi kosong, bisa melihat pos tetapi tidak bisa ganggu, bisa membuat tampilan frontend, akan tetapi tidak muncul langsung dan harus di setujui apakah bagus atau tidak
  12. pada saat kita membuat pengguna baru jika peranannya adalah penulis, maka orang tersebut bisa mengubah atau menambahkan sesuatu didalam frontend, tanpa harus di tinjau kembali
  13. pada saat kita membuat pengguna baru jika peranannya adalah editor, maka orang tersebut bisa mengubah atau menambahkan sesuatu didalam frontend, editor juga tangan kanan admin, dan bisa langsung muncul apa yang di buatkan di editor tanpa persetujuan admin, tetapi tidak ada menu atau instal plugin dan tampilan
  14. pada saat kita membuat pengguna baru jika peranannya adalah administator, bisa mengintal plugin dan tampilan dan posisinya yang paling tinggi dari sebelumnya dan bisa mengubah dan menghapus
  15. di menu pengaturan terdapat didalamnya seperti : umum(untuk merubah judul, merubah waktu, tanggal, menulis(untuk mangatur format posnya), membaca(untuk pos terbaru agar langusng muncul di beranda ), diskusi(untuk berdiskusi komentar yang muncul), media(untuk melohat apa saja yang di upload atau gambar yang di upload, terdapat beberapa ukuran gambar pada folder website kita yang optimal), permalink(untuk mengatur struktur link dengan berbagai bentuk atau format), privasi(untuk menyimpan data lokasi agar tidak di akeses atau di hack)
